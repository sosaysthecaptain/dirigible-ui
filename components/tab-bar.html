Tab Bar

A narrow vertical strip of icon buttons used as top-level navigation when the Sidebar is collapsed. Each button represents a section of the app. The active state uses inverted colors — var(--fg) background with var(--bg) icon.

When to Use

  • Collapsed sidebar state — replaces the full sidebar with icon-only navigation
  • Top-level section switching (Recents, Bookmarks, Docs, Albums, Music)
  • Utility actions pinned to the bottom (Themes, Mode toggle, Sync)

Variants

State Description
DefaultIcon in var(--muted), transparent background
HoverIcon becomes var(--fg), background var(--hover)
ActiveInverted: var(--fg) background, var(--bg) icon color

Anatomy

<div class="tab-bar">                            <!-- container -->
  <div class="tab-bar-section">                  <!-- top: navigation -->
    <button class="tab-bar-btn active">          <!-- active item -->
      <svg ... />                                <!-- 14x14 icon -->
    </button>
    <button class="tab-bar-btn">...</button>
  </div>
  <div class="tab-bar-spacer" />                 <!-- flex:1 spacer -->
  <div class="tab-bar-section">                  <!-- bottom: utilities -->
    <button class="tab-bar-btn">...</button>
  </div>
</div>

Do's and Don'ts

Do

  • Use inverted colors (fg background, bg icon) for the active state
  • Group navigation icons at the top and utility icons at the bottom
  • Use a flex spacer to push utility icons to the bottom
  • Keep icons at 14x14 — the 40px bar is deliberately compact
  • Add title attributes for accessibility since there are no visible labels

Don't

  • Don't use text labels — the tab bar is icon-only by design
  • Don't add dividers between sections — spacing alone groups them
  • Don't use accent color for active state — inverted fg/bg is the pattern
  • Don't use the tab bar when the full sidebar is visible

Rules

  1. Container is exactly 40px wide. Background is var(--sidebar-bg).
  2. Active button uses inverted colors: background: var(--fg), color: var(--bg).
  3. Only one button can be active at a time.
  4. No separators — vertical spacing and the flex spacer create visual grouping.
  5. Icons are 14x14, consistent with the Dirigible icon system at this size tier.
  6. The tab bar replaces the full sidebar when collapsed — they are never both visible.
